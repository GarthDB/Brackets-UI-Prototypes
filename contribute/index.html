Brackets is an open source code editor built in HTML, CSS, and JavaScript. It is designed from the ground up to make it easy for web developers, like you, to hack on it and to write extensions.

We’ve rounded up all the information you’ll need to get started and organized it on this handy site.

Getting Started

Get the Source

So to start, you'll need Brackets, and you can find it on Github. Take a look a the code, fork it, pull a clone, whatever you want.
