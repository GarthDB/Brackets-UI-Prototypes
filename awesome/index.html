Cordova

  • @apachecordova
  • cordova.apache.org
A set of device APIs that allow mobile app developer to access native device functions such as camera or accelerometer from JavaScript to build a smartphone app using just HTML, CSS & JavaScript.

How to contribute

There are many avenues to contribute to Cordova! You can start by looking at the JIRA issue tracker or browsing the mailing list or learn more about contributing to specific platforms.

Webplatform.org

  • @webplatform
  • webplatform.org
Web Platform Docs is a community-driven site that aims to become a comprehensive and authoritative source for web developer documentation.

How to contribute

Start making the documentation better by editing existing content, filing bugs or feature requests, or joining the mailing list.

Test the Web Forward

  • @testthewebfwd
  • testthewebforward.org
Test the Web Forward is an event where you learn about how to write tests for Open Web features & contribute tests, file bugs on browsers to make the web better.

How to contribute

Come by to an event near you or join the mailing list to learn more!
